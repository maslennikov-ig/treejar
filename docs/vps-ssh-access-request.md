## Часть 2: Что могло утечь и что нужно поменять

На взломанном сервере находился файл `.env` со всеми секретами. **Считаем их все скомпрометированными.**

### Инструкция для владельца: ротация ключей

Пожалуйста, замени следующие ключи и пришли новые **одним сообщением** в формате ниже.

---

### 1. 🔴 OpenRouter API Key
**Что это:** Ключ доступа к LLM (GPT, Claude и т.д.) через OpenRouter.
**Где поменять:** https://openrouter.ai/keys → удали старый ключ, создай новый.
**Утёкший ключ начинается с:** `sk-or-v1-31f7...`

### 2. 🔴 Wazzup API Key
**Что это:** Ключ доступа к WhatsApp Gateway (отправка/получение сообщений).
**Где поменять:** https://wazzup24.com → Настройки → Интеграции → API → пересоздать ключ.
**Утёкший ключ начинается с:** `b5132bb0...`

### 3. 🔴 Zoho CRM / Inventory OAuth Credentials
**Что это:** Доступ к CRM (контакты, сделки) и складу (товары, цены).
**Где поменять:** https://api-console.zoho.eu → Self Client → сгенерировать новый Client Secret и Refresh Token.
**Нужно обновить:**
- `Client Secret` (начинается с `b121df09...`)
- `Refresh Token` (начинается с `1000.cc68f8b6...`)
- `Client ID` — можно оставить, если секрет обновлён

### 4. 🔴 Telegram Bot Token
**Что это:** Управление Telegram-ботом для уведомлений менеджеру.
**Где поменять:** @BotFather в Telegram → `/revoke` → получить новый токен.
**Утёкший токен начинается с:** `8651031074:AAG5...`

### 5. 🟡 VPS SSH Key (GitHub Actions)
**Что это:** SSH-ключ для автодеплоя.
**Действие:** Сгенерировать новую SSH-пару на новом сервере и обновить GitHub Secret `VPS_SSH_KEY`.
```bash
ssh-keygen -t ed25519 -f ~/.ssh/treejar_deploy -N ""
cat ~/.ssh/treejar_deploy.pub >> ~/.ssh/authorized_keys
# Содержимое ~/.ssh/treejar_deploy → в GitHub Secret VPS_SSH_KEY
```

### 6. 🟡 SSL-сертификат (Let's Encrypt)
**Действие:** Будет автоматически перевыпущен через Certbot на новом сервере.

---

### Формат для отправки новых ключей

Пожалуйста, пришли всё **одним сообщением** в таком формате:

```
OPENROUTER_API_KEY=sk-or-v1-...
WAZZUP_API_KEY=...
ZOHO_CRM_CLIENT_SECRET=...
ZOHO_CRM_REFRESH_TOKEN=...
TELEGRAM_BOT_TOKEN=...
VPS_SSH_KEY=(приватный ключ отдельным файлом)
Новый IP сервера: x.x.x.x
```

> ⚠️ **ВАЖНО:** Пока ключи **не поменяны**, злоумышленник потенциально может:
> - Читать/отправлять WhatsApp-сообщения от имени бота
> - Получить доступ к Zoho CRM (контакты клиентов, сделки)
> - Слать сообщения через Telegram-бота
> - Использовать LLM за ваш счёт (OpenRouter)
>
> **Рекомендуется поменять ключи как можно скорее.**
