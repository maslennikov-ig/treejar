## Выводы для обучения Noor

**Делать как в успешных:**
1. Быстро отправлять фото + цену товара
2. Предлагать несколько вариантов (разный дизайн, цена)
3. После согласия — сразу спрашивать данные для квоты
4. Предлагать скидку/бонус при комплексном заказе
5. Организовывать доставку и follow-up

**Избегать:**
1. Затянутые диалоги без перехода к квоте
2. Отсутствие альтернатив при нехватке товара на складе
3. Потеря клиента после торга без закрытия
