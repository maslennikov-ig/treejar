## 3. Источники данных о товарах

**Статус:** Закрыто 2026-04-06

- Канонический источник для Noor: `https://new.treejartrading.ae/api/catalog`
- `bazara.ae` и старый `treejartrading.ae/ksa-en` не используются как источники истины по каталогу
- От заказчика дополнительно нужен доступный `docs/catalog-api.md` или эквивалентный контракт API

**Зачем это зафиксировано:** это снимает двусмысленность по источникам каталога и позволяет строить Noor вокруг одного канонического product API.
