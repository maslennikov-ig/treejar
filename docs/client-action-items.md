## Шаги регистрации (пошагово)

### Supabase Cloud
1. Перейти на https://supabase.com
2. Sign Up (через GitHub или email)
3. New Project → выбрать Organization
4. Region: **EU (Frankfurt)** или **Singapore**
5. Придумать Database Password (сохранить!)
6. Передать нам: Project URL, anon key, service_role key, Database Password

### OpenRouter
1. Перейти на https://openrouter.ai
2. Sign Up
3. Settings → API Keys → Create Key
4. Billing → Add Credits ($10-20)
5. Передать нам: API Key

### VPS (мы подберём)
1. Мы пришлём ссылку на конкретный тариф (Hetzner/DigitalOcean)
2. Клиент регистрируется и оплачивает
3. Передаёт нам SSH-доступ (или мы регистрируем на своё имя, клиент возмещает)
