## Структура КП (общая для всех)

Каждое КП содержит:

**Шапка:**
- Логотип Skyland + Treejar Trading FZC LLC
- Номер квоты (формат: `#XX DDMMYY - RN`, где XX — инициалы, R — ревизия)
- TRN (Tax Registration Number)
- Дата

**Данные клиента:**
- Name / Company Name / Email / Address

**Таблица товаров:**
| Колонка | Описание |
|---------|----------|
| Reference Image | Фото товара (реальное или каталожное) |
| Code/SKU | Код модели (например: NOVO 1600, CH 145 GREY) |
| Description | Полное описание: материалы, размеры, цвет, гарантия |
| QTY | Количество |
| Unit Price | Цена за единицу (AED) |
| Total Price | Итого за позицию (AED) |

**Итого:**
- TOTAL (сумма без НДС)
- VAT 5%
- GRAND TOTAL

**Terms and Conditions:**
- Brand: Skyland
- Delivery: 02-08 working days from order confirmation
- Payment Terms: 50% advance, 50% after installation
- Validity: 15 days from date of quotation and/or till stock last
- Amendment: no alterations after PO issued
- Pictures: illustration only, actual may vary
- Quoted by: имя + телефон + email менеджера

## Выводы для Noor: как формировать КП

1. **Обязательные данные:** имя клиента, компания, email, адрес
2. **Фото каждого товара** — реальное или каталожное
3. **Полное описание:** материалы, размеры (см), цвет, гарантия
4. **Для крупных заказов:** структурировать по отделам/зонам (как CH 090226)
5. **Для каждой зоны:** полный комплект (стол + тумба + стул + шкаф)
6. **Delivery включать отдельной строкой** (free для крупных заказов)
7. **VAT 5%** всегда указывать отдельно
8. **Terms & Conditions** стандартные в подвале
9. **Ревизии (R1, R2...)** — дорабатывать КП под клиента
10. **Предлагать 2+ варианта** (разные модели, разные цены, разные бренды)
