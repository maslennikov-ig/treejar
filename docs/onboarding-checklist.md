## 1. Доступы и API-ключи (до старта)

Без этих доступов мы не сможем начать. Пожалуйста, подготовьте до подписания договора.

| # | Что нужно | Как получить | Зачем |
|---|-----------|-------------|-------|
| 1.1 | **Zoho CRM — API-ключ** | Zoho Developer Console → Self Client → Client ID + Client Secret + Refresh Token | Создание контактов, сделок, UTM-меток, история клиента |
| 1.2 | **Zoho Inventory — API-ключ** | Аналогично Zoho CRM (может быть тот же Client ID, если подключён единый Zoho One) | Товары, цены, остатки, создание SaleOrder |
| 1.3 | **Wazzup — API-ключ** | Личный кабинет Wazzup → Настройки → API | Приём и отправка сообщений в WhatsApp |
| 1.4 | **GitHub аккаунт** | Создать репозиторий (Owner) и пригласить исполнителя как Collaborator | Код проекта хранится у вас |
| 1.5 | **URL обоих сайтов компании** | Прислать ссылки | Парсинг информации для базы знаний |

**Как передать ключи безопасно:** через приватное сообщение или любой менеджер паролей (1Password, Bitwarden). Не через обычный email или мессенджер.
